# Page Nouvelle intervention - Documentation

## Vue d'ensemble

Création d'une nouvelle intervention.

- Route: `/intervention/new`
- ID: `interventions-new`
- Composant: `InterventionCreate`
- Accès: Privé (auth requis)
- Menu: masquée (`showInMenu: false`), icône `Plus`

## En-tête

- Titre: "Créer une intervention"
- Sous-titre: "Ajouter une nouvelle intervention"

## Notes
