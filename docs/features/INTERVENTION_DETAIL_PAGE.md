# Page Détail intervention - Documentation

## Vue d'ensemble

Consultation détaillée d'une intervention.

- Route: `/intervention/:id`
- ID: `intervention-detail`
- Composant: `InterventionDetail`
- Accès: Privé (auth requis)
- Menu: masquée (`showInMenu: false`), icône `FileText`

## En-tête

- Titre: "Détail de l'intervention"
- Sous-titre: dynamique selon l'ID

## Notes

- Accès depuis la liste des interventions ou liens internes.
